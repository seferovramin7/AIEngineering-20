# Generative Models Explained

This is an interactive web application to explain the concepts of Generative Adversarial Networks (GANs) and Variational Autoencoders (VAEs).

## How to Run

1.  Open the `index.html` file in your web browser.
2.  Click on the "Run VAE" and "Run GAN" buttons to see the simulated output of the models.
